# Premier League Ranking Scraper (R)

This project uses R to scrape **team rankings and names** from the official Premier League standings page. The data is saved as a CSV for later use in data analysis.

## 🛠 Technologies Used

- R
- `rvest` – for HTML parsing and data extraction
- `readr` – for saving scraped data to CSV

## 🚀 How to Use

1. Open `scrape_table.R` in RStudio or another R environment
2. Run the script
3. Output CSV file will be saved to `data/standings.csv`

## 📎 Example Output

| Rank | Team Name        |
|------|------------------|
| 1    | Arsenal          |
| 2    | Manchester City  |
| 3    | Liverpool        |
| ...  | ...              |
